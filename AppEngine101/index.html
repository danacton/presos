Some important concepts

Datastore

  • Schemaless NoSQL database
  • Object/Entity persistence
  • Limited to 1MB per entity

Blobstore

  • Store arbitrary data (e.g. images or files)
  • No size limit

Other storage options

  • Google Cloud SQL: relational database in the cloud
  • Google Cloud Storage: similar to a cloud-based filesystem

Some important concepts

Use Memcache

  • Key-value in-memory cache
  • Before going to database, check the cache
  • Put often-used data in the cache

Defer (with Tasks) where you can

  • time(task) > x ms => defer
  • Example: lazy write, quick read

Getting started with Google App Engine

Part 2: A Hello World project in Python

Create a file helloworld/helloworld.py

import webapp2

class MainPage(webapp2.RequestHandler):

    def get(self):
        self.response.headers['Content-Type'] = 'text/plain'
        self.response.write('Hello, World!')

application = webapp2.WSGIApplication([
    ('/', MainPage),
], debug=True)
